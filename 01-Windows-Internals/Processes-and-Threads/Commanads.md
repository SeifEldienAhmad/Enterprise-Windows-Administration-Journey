```
tasklist: displays a list of all currently running applications, background services, and processes on a Microsoft Windows operating system
```

```
taskkill:  used to end one or more running tasks or processes
```

```
resmon: launches Windows Resource Monitor, an advanced diagnostic tool. It provides real-time, in-depth data on your system's hardware resource usage across four main categories: CPU, Memory, Disk, and Network
```

```
Get-Process: cmdlet in PowerShell retrieves information about the active, running processes on a local or remote computer
```
