# Enterprise Windows Administration Journey

> A public portfolio documenting my journey toward becoming an Enterprise IT Support and System Administration Engineer through hands-on labs, technical documentation, and real-world troubleshooting scenarios.

---

## About

This repository is a structured knowledge base that documents my practical journey in Windows Administration, System Administration, Networking, PowerShell, Active Directory, and Microsoft 365.

The primary goal is not to complete courses, but to understand how enterprise IT environments work by building labs, documenting findings, troubleshooting real-world scenarios, and continuously improving technical skills.

Every topic in this repository includes practical exercises, technical notes, screenshots, commands, and lessons learned.
